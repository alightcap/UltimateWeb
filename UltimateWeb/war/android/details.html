UltiAnalytics

UltiAnalytics is an Android phone/tablet app and web site for tracking statistics for your ultimate team(s). Users create one or more teams in the app and gather stats as the game is in progress. When the game is complete the user can upload the statistics to their team website (provided). The app also features a Twitter interface that allows the user to "auto-tweet" game events as they happen.

Download

Features

  • Create one or more teams that can be uploaded and downloaded to different Android phones/tablets or iPhones/iPads (for backup or passing on the tracking duties...or a low battery situation).
  • Gather stats while the game action progresses
    1. passes, drops, assists, goals, d's, throwaways with additional detail as well (see screenshots)
    2. move players from field to bench with help on gender and playing time
    3. all game events available to view and undo
    4. wind effect
  • Upload games(s) to your team site (each team you create get it's own statistics website) after the game. Example: Sample Web Site
  • Tweet the progress of the game for those that can't attend. Use "auto-tweet" to have the app tweet progress for you and/or use the integrated Tweet view to send ad-hoc tweets.
